# Blog

Це репозиторій блогу де кожний може ввести свій блог\
Посилання на блог <https://semi-blog.fun/>

## Початок роботи

```sh
npm install
```

```sh
npm run build
```

```sh
cd Server/build
```

```sh
npm run start:migrate:prod
```
